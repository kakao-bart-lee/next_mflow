## 2. 사주(四柱) 엔진 — 전체 출력 데이터

### 2.1 고정 데이터 (출생 시 결정, 영원 불변)

#### 사주 4기둥 (1-1) — 현재 일주만 표시

```
년주: 천간(甲~癸) + 지지(子~亥) + 천간오행 + 지지오행
월주: 천간 + 지지 + 천간오행 + 지지오행
일주: 천간 + 지지 + 천간오행 + 지지오행  ← 현재 이것만 표시
시주: 천간 + 지지 + 천간오행 + 지지오행

즉시 표시 가능: 4기둥 전체를 시각화
→ 년주(부모/조상운) · 월주(사회운) · 일주(자신/배우자) · 시주(자녀/말년)
```

#### 십이운성 (1-2) — 미표시

```
각 기둥별: 장생·목욕·관대·건록·제왕·쇠·병·사·묘·절·태·양 중 하나

즉시 표시 가능: 4기둥의 에너지 상태를 시각적 그래프로
→ "년주: 제왕(최고조), 월주: 건록(안정), 일주: 장생(시작), 시주: 관대(성장)"
→ 인생 에너지 커브로 시각화
```

#### 지장간 (1-3) — 미표시

```
각 지지 안에 숨은 천간 (1~3개) + 각각의 십신

즉시 표시 가능: 숨은 영향력을 드러내는 UI
→ "일지 속에 정재·편인·겁재가 숨어있습니다"
→ 배우자궁(일지) 분석에 핵심
```

#### 신살 (1-4) — 미표시

```
도화살, 역마살, 화개살, 귀문관살, 학당귀인, 천을귀인 등

즉시 표시 가능: 태그/뱃지 UI
→ 🌸 도화살 (매력·인연)
→ 🐎 역마살 (이동·변화)
→ 🙏 화개살 (예술·영적 감수성)
→ 각 신살의 의미를 카드 형태로
```

#### 십신 분포 (1-5) — 미표시

```
비겁(n), 식상(n), 재성(n), 관성(n), 인성(n)
dominant_sipsin: 가장 강한 십신

즉시 표시 가능: 5축 레이더 차트 + dominant 하이라이트
→ "당신의 에너지 유형: 식상형 (표현·창조)"
→ 비겁=독립/경쟁, 식상=표현/창조, 재성=현실/재물, 관성=질서/책임, 인성=학습/보호
```

#### 십신 위치 (1-6) — 미표시

```
년간·월간·시간·년지·월지·일지·시지 각각의 십신명

즉시 표시 가능: 4기둥에 십신 오버레이
→ 년간=편관(사회적 도전), 월간=정인(학습·보호), 시간=식신(표현·재능)
→ 어느 기둥에 어떤 에너지가 있는지 한눈에
```

#### 오행 파워 (1-7) — ✅ 현재 표시 중

```
목(n), 화(n), 토(n), 금(n), 수(n)

현재: 5축 바 차트로 표시
개선: 부족 원소 하이라이트 + 보충 방법 연결
```

#### 신강/신약 (1-8) — ✅ 현재 표시 중

```
strength_type: 신강/신약
strength_score: 수치

현재: 텍스트로 표시
개선: 게이지 UI + 해석 텍스트
```

#### 형충파해 (1-9) — 미표시 ⭐ 중요

```
삼합: [{관련 지지들}]
천간합: [{관련 천간들}]
삼형살: [{관련 지지들}]
자형살: [{관련 지지들}]
육충살: [{관련 지지들}]
육파살: [{관련 지지들}]
육해살: [{관련 지지들}]

즉시 표시 가능: 관계 다이어그램
→ 기둥 간 합(조화) / 충(갈등) / 형(긴장) 관계를 시각적 선으로 연결
→ "월지-일지 충(冲) → 사회생활과 개인생활 사이 긴장"
→ "년지-일지 삼합 → 조상운과 자신의 에너지가 조화"
```

### 2.2 동적 데이터 (시간에 따라 변함)

#### 대운 (1-10, 1-11) — ✅ 현재 표시 중

```
periods[]: 각 대운의 시작/종료 나이, 천간·지지·십신
current_period: 현재 대운

현재: 타임라인 표시
개선: 각 대운 기간에 해석 텍스트 + 십신 의미 + 다른 시스템 타이밍 오버레이
```

## 3. 점성술(Astrology) 엔진 — 전체 출력 데이터

### 3.1 고정 데이터 (출생 차트)

#### 행성 위치 (2-1) — ✅ 현재 표시 중

```
7개 행성: Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn
각각: lonDeg(황경), sign(별자리), signDeg(별자리 내 도수), house(하우스)

현재: PLANET_META 카드로 표시 (별자리 + 하우스 + 영향력)
개선: 행성별 해석 텍스트 + 사주 십신 연결 (PLANET_META에 이미 매핑 존재)
```

#### 어스펙트 (2-2) — 부분 표시

```
aspects[]: 두 행성 간 각도 관계
  type: conjunction(합)/opposition(충)/trine(삼합)/square(사각)/sextile(육각)
  orb: 허용도 (좁을수록 강함)
  applying: 가까워지는 중(강화) vs 멀어지는 중(약화)

현재: 트랜짓 카드에서 일부 표시
개선: 출생 차트 어스펙트 전체 시각화 (원형 차트에 선 그리기)
```

#### ASC/MC (2-3, 2-4) — ✅ 현재 표시 중

```
ASC: 상승점 별자리 + 도수
MC: 중천 별자리 + 도수

현재: 텍스트 표시
개선: 의미 해석 추가 ("ASC 사자자리 = 외부에 보이는 모습이 당당하고 카리스마")
```

#### 12하우스 배치 (2-5) — 간접 표시

```
houses[]: 각 하우스의 경계(cusp) 도수 + 별자리

현재: 행성-하우스 배치로 간접 표시
개선: 하우스 차트 시각화 (12칸 원형/사각 차트)
```

#### 낙샤트라 (2-7, 2-8) — 표시 중

```
각 행성의 nakshatra: name, pada, ruling_planet, degree
moonNakshatra: 달의 낙샤트라 (Vedic 핵심)

현재: 행성별 표시
개선: 달 낙샤트라 특별 카드 ("당신의 달 낙샤트라: Rohini — 창조와 풍요")
```

#### 주야 차트 (2-11) — 미표시

```
isDayChart: boolean (태양이 수평선 위인지)

즉시 표시 가능: "당신은 Day Chart입니다 ☀️"
→ Sect 분석의 기초 (주간 차트는 Sun/Jupiter/Saturn이 유리, 야간은 Moon/Venus/Mars)
```

### 3.2 새 API — 미연결 고급 데이터 ⭐

#### Essential Dignity (4-2) — 라우트 존재, 미호출

```
각 행성의 전통적 품위:
  domicile (본거지): +5
  exaltation (고양): +4
  triplicity (삼주성): +3
  term (바운드): +2
  face (데칸): +1
  detriment (실추): -5
  fall (함락): -4
  peregrine (편력): -5

즉시 표시 가능: 행성별 "품위 카드"
→ "금성: Taurus에서 domicile ★★★★★ — 금성이 가장 편안한 자리"
→ "화성: Cancer에서 fall ☆ — 화성 에너지가 약해지는 배치"
→ 색상 코딩 (초록=강, 빨강=약)
```

#### Accidental Dignity (4-3) — 라우트 존재, 미호출

```
하우스 강도 가감점
역행(retrograde) / 직행(direct) / 정지(stationary) 상태
연소(combustion) / cazimi (태양에 너무 가까움)
어스펙트 가감점

즉시 표시 가능: 행성 "상태 뱃지"
→ "수성 ↺ 역행 중" / "금성 ☀️ cazimi (태양과 합일)"
→ 역행 행성 특별 알림 시스템
```

#### 오늘의 인사이트 확장 (4-4) — 부분만 표시

```
현재 미표시:
  dominantPlanet: 오늘의 지배 행성
  tags[]: 오늘의 테마 태그
  actions[]: 오늘의 추천 행동
  caution: 오늘의 주의사항

즉시 표시 가능:
→ "오늘의 지배 행성: ♃ 목성 — 확장과 기회의 에너지"
→ tags를 칩/뱃지로: #확장 #낙관 #과잉주의
→ actions를 체크리스트로: □ 새로운 기회에 눈 돌리기 □ 과도한 약속 주의
→ caution을 경고 배너로
```

#### 미래 인사이트 (4-5) — 미표시

```
future.days[]: 향후 7일 각각
  date, dominantPlanet, theme, focus, intensity(low/medium/high)

즉시 표시 가능: 7일 에너지 그래프
→ 각 날의 intensity를 막대/파형으로
→ "수요일이 이번 주 피크 에너지"
→ 달력 오버레이
```

#### 행성 파워 랭킹 (4-6) — 미표시

```
ranking: 영향력 기준 행성 순위

즉시 표시 가능: "당신의 TOP 3 행성"
→ 1위 ♃ 목성 (87점) — 확장/기회
→ 2위 ♀ 금성 (74점) — 관계/미학
→ 3위 ☉ 태양 (68점) — 자아/리더십
```

#### Sect + Lots (4-8) — 라우트 존재, 미호출

```
Sect:
  sect: DAY / NIGHT
  행성별 sect_score: 주간/야간 차트에서의 유불리

Lots (아라빅 파츠):
  Lot of Fortune: sign + degree + house
  Lot of Spirit: sign + degree + house
  Lot of Eros: sign + degree + house
  Lot of Courage: sign + degree + house
  Lot of Necessity: sign + degree + house

즉시 표시 가능: "운명의 포인트" 카드
→ 🍀 행운의 점(Lot of Fortune): 쌍둥이자리 15° (3하우스)
→ 🔥 영혼의 점(Lot of Spirit): 전갈자리 22° (8하우스)
→ 💕 사랑의 점(Lot of Eros): 물고기자리 8° (12하우스)
```

#### Profection (4-9) — 라우트 존재, 미호출 ⭐ 핵심

```
profected_house: 올해 활성화된 하우스 (1~12)
profected_sign: 해당 별자리
time_lord: 올해의 지배 행성

## 6. 크로스시스템 융합 데이터 — LLM 없이 계산 가능

> 아래는 3개 시스템의 데이터를 결정적(deterministic)으로 교차 비교해서 새로운 정보를 생성하는 것.

### 6.1 기본 매핑 (이미 코드에 존재하는 것)

#### 일간 오행 ↔ 지배 행성 (6-1)

  金(Metal) ↔ 금성(Venus)
  水(Water) ↔ 수성(Mercury)
  木(Wood) ↔ 목성(Jupiter)
  火(Fire) ↔ 화성(Mars)

비교 결과:
  일치 → "동서양이 같은 원소를 가리킵니다 — 강한 정체성"
  불일치 → "동서양이 다른 면을 봅니다 — 다면적 성격"
```

#### 십신 ↔ 행성 매핑 (6-2) — PLANET_META에 이미 존재

```
현재 코드의 PLANET_META:
  SUN = 식신 (표현/재능)
  MOON = 편인 (직관/보호)
  MERCURY = 정인 (학습/지식)
  VENUS = 정재 (안정/재물)
  MARS = 편관 (도전/변화)

  SATURN = 편재 (유동/투자/모험)

이미 프론트에 존재하지만 활용이 미미
→ 행성 카드에서 "이 행성은 사주의 OO 에너지와 연결됩니다" 표시 가능
```

### 6.2 타이밍 교차 비교 (6-3, 6-4) — 새로 계산 필요

#### 대운 ↔ Profection 비교 (6-3)

```
사주 대운의 십신 방향 vs Profection의 활성 하우스 방향

계산:
  대운 십신이 "재성" + Profection이 "2H(재물)" → "동서양 모두 재물 시기"
  대운 십신이 "관성" + Profection이 "10H(커리어)" → "동서양 모두 커리어 시기"
  대운 십신이 "인성" + Profection이 "9H(학습)" → "동서양 모두 학습 시기"

표시: "올해의 동서양 일치도"
→ 같은 방향이면: 🟢 "사주와 점성술이 같은 테마를 가리킵니다"
→ 다른 방향이면: 🔵 "사주는 OO, 점성술은 OO를 강조합니다"
```

#### 대운 ↔ Dasha 비교 (6-4)

```
사주 10년 대운 ↔ 베다 Maha Dasha 기간

### 6.3 충돌/시너지 자동 감지 (6-5, 6-6) — 새로 계산 필요

#### 충돌 감지 (6-5) — "양쪽 다 경고하는 날"

```

감지 로직:
  1. 오늘의 일주와 원국 지지 사이에 충(冲)이 있는가?
  2. 오늘의 트랜짓에 opposition 또는 square가 있는가?
  3. 둘 다 YES → 🔴 충돌 알림

표시: 달력 위의 빨간 마커 + 주의 배너
→ "3/15: 사주 일지-월지 충 + 화성-토성 오포지션 — 갈등 에너지 주의"
```

#### 시너지 감지 (6-6) — "양쪽 다 좋다는 날"

```

감지 로직:
  1. 오늘의 일주와 원국 사이에 삼합/천간합이 있는가?
  2. 오늘의 트랜짓에 trine 또는 conjunction이 있는가?
  3. 둘 다 YES → 🟢 시너지 알림

표시: 달력 위의 초록 마커 + 기회 배너
→ "3/18: 사주 삼합 형성 + 금성-목성 트라인 — 관계와 재물 모두 좋은 날!"
```

  시너지 감지(7-6) + 자미두수 일운 록성
→ 달력에 "하기 좋은 날" 하이라이트
→ 🟢 = 시너지 날, 🔴 = 충돌 날, ⚪ = 보통 날
```

#### 7일 에너지 그래프 (8-3)

```
future.days[].intensity + 사주 일간 에너지 점수
→ 7일 에너지 곡선
→ "수요일이 이번 주 피크"
```
