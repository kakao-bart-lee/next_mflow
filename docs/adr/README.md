# Architecture Decision Records

이 디렉토리는 프로젝트의 주요 아키텍처 결정을 기록합니다.

## 포맷

각 ADR은 다음 구조를 따릅니다:

- **Context**: 결정이 필요했던 배경과 문제
- **Decision**: 선택한 해결책
- **Consequences**: 결정의 결과 및 트레이드오프
